## Istruction
VERSIONE DI TEST: 
```
git clone https://gitlab.com/made-4ci/made-website.git
cd made-website
npm install
ng serve
```
COMPILAZIONE DEL PROGETTO:
```
git clone https://gitlab.com/made-4ci/made-website.git
cd made-website
npm install 
ng build
```
